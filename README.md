# Mamo Site V2 - Frontend

هذه هي الواجهة البرمجية لمشروع **Mamo Site**، وهي مبنية باستخدام تقنيات الويب الحديثة (HTML5, CSS3, JavaScript) ومرتبطة مباشرة بقاعدة بيانات **Supabase**.

## 🚀 كيفية التشغيل محلياً
1. قم بفتح مجلد `MamoSite_Web`.
2. قم بنسخ ملف اللوجو `mamo.png` من مجلد `TelegramBot_V2/Info` إلى هذا المجلد.
3. افتح ملف `index.html` في أي متصفح.

## 🌐 كيفية الرفع مجاناً (Hosting)
يمكنك رفع هذا الموقع خلال ثوانٍ وبشكل مجاني تماماً عبر منصة **Netlify** أو **Vercel**:

### عبر Netlify (الأسهل):
1. اذهب إلى [Netlify Drop](https://app.netlify.com/drop).
2. قم بسحب وإسقاط مجلد `MamoSite_Web` بالكامل إلى المتصفح.
3. مبروك! سيعطيك الموقع رابطاً مباشراً لموقعك.

### عبر Cloudflare Pages (الأفضل للمستقبل):
1. اربط حسابك في GitHub بـ Cloudflare.
2. اختر مستودع المشروع وحدد مجلد `MamoSite_Web` كمجلد البداية.
3. سيتم الرفع تلقائياً مع كل تحديث للكود.

## ✨ المميزات
- **Real-time Search**: بحث فوري في المواقع.

- **Responsive**: متوافق تماماً مع جميع شاشات الجوال والحواسيب.
